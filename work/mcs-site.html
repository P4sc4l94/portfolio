Memory Care Summit

2021

This was a personal project I did while I was redesigning the Memory Care Summit materials. I wanted to have a website that reflected the professionalism of the Summit without the constraints of UMBC's site template.

View MCS Site View Original MCS Website

Initial Concept

After sketching initial page layout ideas, I brought those over into Photoshop and created various layouts.

Final Version

I cleaned up the layout: added more whitespace, darkened the colors for visibility, and settled on a color palette. I also rearranged sections so they are able to adjust fluidly to different viewports. Once satisfied, I built the website in Visual Studio and used Boostrap to assist with structuring the pages.
